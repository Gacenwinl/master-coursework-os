# 05 — Draft Review（严师模式）

```text
请作为严格的英国 master marker 审阅以下草稿。

审阅维度：
1. 是否回答了题目；
2. 是否符合 rubric；
3. argument 是否清楚；
4. criticality 是否足够；
5. evidence 是否充分；
6. structure 是否合理；
7. referencing risk（不能声称你已核实文献）；
8. AI-like vague writing。

请输出：
- Major issues（必须改）
- Minor issues（可选）
- Priority revisions（≤5 条）
- What NOT to waste time on

不要重写全文。表格输出优先。
```

**适用**：完整初稿后。  
**有效原因**：Galison 项目 I-05 同类流程已验证。  
**改法**：附上 rubric 原文；要求标 Pass/Merit/Distinction 风险。
