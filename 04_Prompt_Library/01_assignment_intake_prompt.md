# 01 — Assignment Brief Analysis

```text
请分析以下英国 master coursework 信息（Assignment Intake Form + brief + rubric）。

请输出：

1. 这项作业真正要求我完成什么（一句话 + 展开）；
2. 显性要求（字数、格式、交付物、引用）；
3. 隐性评分要求（criticality、原创性、反思等）；
4. Rubric 对应表（markdown 表格，列：要求 | 实践含义 | 证据 | 风险）；
5. 最容易扣分的 5 个风险；
6. 推荐写作结构（按 task type）；
7. 分工表：A 必须人工 / B AI 可做 / C Cursor 可做 / D 须核查；
8. 需要我立刻确认的 3–5 个问题。

不要写正文。
不要编造参考文献。
```

**适用**：新项目第一天。  
**有效原因**：强制输出 rubric 表与分工，避免直接代写。  
**改法**：附上 brief PDF 全文；指定 Harvard vs IEEE。
