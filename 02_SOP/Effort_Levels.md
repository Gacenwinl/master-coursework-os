# 投入档位（低 / 中 / 高）

与 **Mode A/B/C**（AI 强度）正交：先选 Mode，再选档位。  
**Target = Distinction** → 读 `Distinction_Target_SOP.md`，默认 **High** 档。

| 档位 | 总时长 | 适用 | 裁剪 |
| --- | ---: | --- | --- |
| **Low** | 8–12h | 15% 小 essay；复用熟悉理论 | 单轮 critique；简 literature；无 TikZ |
| **Medium** | 18–25h | 标准 coursework；**Merit 默认** | Galison 实际接近此档 |
| **High** | 35–50h | Distinction 目标；dissertation 章；双交付 | 五关卡 + 多轮 critique |

Low：Draft↑、Literature↓。High：见下表。

## High / Distinction 时间分配（%）

| Phase | % | 说明 |
| --- | ---: | --- |
| Intake + Module0 + rubric | 10% | 含 Distinction 目标确认 |
| Literature + notes + **关卡 B** | 28% | 争论文献、primary 配额 |
| Argument + **关卡 A** + outline | 22% | thesis_objection_worksheet |
| Draft（**你写为主**） | 20% | Mode B；少 AI 整段 |
| Critique + **关卡 C/D** | 22% | `15` red team + `05` + 去模板段 |
| Format + **关卡 E** + final | 8% | oral prep + distinction_checklist |

**原则**：Distinction 边际分在 **改论点**，不在多一轮 LaTeX。
