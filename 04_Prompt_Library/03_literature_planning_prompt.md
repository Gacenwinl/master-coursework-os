# 03 — Literature Strategy（不生成具体文献）

```text
请基于以下 coursework topic 和中心论点，设计文献检索策略。

要求：
1. 不要编造任何具体文献条目；
2. 只给：关键词（中英文）、推荐数据库、文献类型；
3. 区分：required reading / core literature / supporting literature；
4. 每类建议篇数范围；
5. 哪些论点必须有引用支撑；
6. 常见引用风险（二手综述、错误年份、领域不符）。

输出表格 + 7 天检索计划（若我提供总小时数）。
```

**适用**：Phase 文献启动。  
**有效原因**：把「找文献」从 AI 幻觉中剥离。  
**改法**：贴上 module reading list 让 AI 只排序优先级，不新增书名。
