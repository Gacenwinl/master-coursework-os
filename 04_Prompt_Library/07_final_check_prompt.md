# 07 — Final Submission Check

```text
附件/粘贴：① 官方 assignment brief（或 rubric）；② 我的终稿 PDF/全文。

请做 submission readiness check，表格输出：
- 指南条款 | 判断（满足/部分/不满足/无法确认）| 证据（页码/摘录）| 修改建议

规则：
- 以 brief 为准，不要自增扣分项；
- 不能声称已核实文献或原著；
- 看不清的写「无法从文件确认」。

最后输出：
1. 必须修改（≤5）；
2. 可改可不改；
3. 是否建议提交；
4. 只能由我完成的终检项清单。
```

**适用**：提交前 24h。  
**有效原因**：Galison 终检 caught Index Terms、Abstract 缩略语。  
**改法**：第三轮只复核「已修改项」。
