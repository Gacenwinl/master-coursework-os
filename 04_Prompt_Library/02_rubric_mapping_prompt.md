# 02 — Rubric Mapping

```text
请把以下 assignment brief 和 marking criteria 转换成 rubric mapping table。

表格列：
- Rubric requirement
- What it means in practice
- Evidence needed in my submission
- Where to address it (section/file)
- Risk if ignored
- My current status (Not started / In progress / Done / Needs human check)

请输出可直接粘贴到 rubric_mapping_table.md 的 markdown 表格。

不要写正文。不要编造文献。
```

**适用**：Intake 填完后、写作前。  
**有效原因**：与评分标准逐条对齐。  
**改法**：要求按 rubric 原文顺序排列。
