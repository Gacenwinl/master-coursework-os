# 04 — Outline / Structure

```text
请为以下作业设计详细大纲（不要写完整正文）。

输入：
- 中心论点（我已写）：
- 字数/页数：
- 必须使用的理论/案例：
- Rubric 重点：

输出：
1. 节标题 + 每节 2–3 句核心论点；
2. 每节需要的证据类型（文献/数据/案例）；
3. 篇幅分配（字数或页数）；
4. 哪里放直接引、哪里转述；
5. Discussion/局限段写什么。

限制：不编造文献；不替我改中心论点。
```

**适用**：文献 Verified 一部分后。  
**有效原因**：控制结构先于 prose。  
**改法**：technical paper 加 Fig/Table 计划。
