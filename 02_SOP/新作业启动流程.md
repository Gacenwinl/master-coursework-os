# 新作业启动流程（一键启动）

> 配合 `Master_Coursework_OS/` 使用。预计 **30–45 分钟**完成启动，不含写作。  
> **英硕**：必须先做 **Step 0**（合规关卡），再进入后续步骤。

---

## Step 0：Module 0 — AI Policy Gate（英硕必做）

1. 阅读 module handbook / assessment brief 中的 **AI 与 academic integrity** 条款  
2. 填写 `02_SOP/Module0_AI_Policy_Gate.md` → 保存到 `01_Brief/Module0_AI_Policy_Gate.md`  
3. 选定 **Mode A / B / C**（见 `UK_Compliance_and_Modes.md`）  
4. 自己写一句：**This assignment is asking me to…**（再让 AI 检查，不让 AI 代写此句）  
5. Gate 未通过 → **不要** 发 ChatGPT 写作类 prompt

---

## Step 1：创建项目文件夹

**方式 A（推荐）**

```powershell
cd Master_Coursework_OS
python create_coursework_project.py
```

按提示输入 module、assignment、deadline、target grade、hours。

**方式 B**

复制 `05_Project_Template/` → 重命名为 `YYYY_SemX_ModuleCode_AssignmentName/`

---

## Step 2：填写 Assignment Intake Form

路径：`01_Brief/Assignment_Intake_Form.md`

**必须填**：deadline、字数、rubric 全文、AI policy、target grade、available hours。

将 brief PDF 放入 `01_Brief/`。

---

## Step 3：Cursor 初始化项目

1. 用 Cursor 打开新项目根目录  
2. 打开并复制 `Master_Coursework_OS/04_Prompt_Library/08_cursor_project_bootstrap_prompt.md` 全文发送  
3. 检查生成文件：`task_plan.md`、`rubric_mapping_table.md`、`literature_table`、`ai_log`、`draft_structure.md`、`final_submission_checklist.md`  
4. 回答 Cursor 列出的「待补充信息」

---

## Step 4：ChatGPT 任务解析

1. 新会话  
2. 先发 `04_Prompt_Library/00_chatgpt_session_opener.md`  
3. 再发 `01_assignment_intake_prompt.md` 或粘贴完整 Intake Form  
4. 保存输出要点到 `02_Notes/brief_analysis.md`（可选）

---

## Step 5：人工确认三件事（不可跳过）

| # | 问题 | 你的答案（写进 `02_Notes/writing_decisions.md`） |
| --- | --- | --- |
| 1 | 这道题到底要回答什么？（一句话） | |
| 2 | 必读文献 / 核心理论是哪几个？ | |
| 3 | AI 使用是否允许？边界是什么？ | |

---

## Step 6：AI 生成工作计划（ChatGPT）

依次使用（可同一会话）：

1. `02_rubric_mapping_prompt.md` → 更新 `03_Literature/../rubric_mapping_table.md`  
2. `03_literature_planning_prompt.md` → 更新检索策略  
3. `04_outline_prompt.md` → 更新 `04_Draft/draft_structure.md`

**禁止**：在此阶段生成可提交的完整正文。

---

## Step 7：文献（人工为主）

按 `Reference_Verification_SOP.md` 填 `03_Literature/literature_table`（csv/md）。

---

## Step 8：写作与审稿

| 阶段 | 工具 | Prompt / 文件 |
| --- | --- | --- |
| 初稿 | 你 + 可选 AI 段落 | `04_Draft/draft.md` |
| 审稿 | ChatGPT | `05_draft_review_prompt.md` |
| 格式 | Cursor | `08_cursor_project_bootstrap` 中的 LaTeX/Word 指令 |

---

## Step 9：终检与提交

1. `07_final_check_prompt.md` + 上传 brief 与 PDF 给 ChatGPT  
2. `Final_Checklist_SOP.md` 逐项勾选  
3. 提交 VLE  
4. （可选）`99_Review/postmortem.md`

---

## 时间分配参考（可调）

| 块 | 占比 | 小时（例：20h） |
| --- | ---: | ---: |
| 读题 + rubric | 10% | 2 |
| 文献 + notes | 20% | 4 |
| outline + 论点 | 20% | 4 |
| draft | 30% | 6 |
| critique + 修订 | 10% | 2 |
| 格式 + 终检 + 提交 | 10% | 2 |
