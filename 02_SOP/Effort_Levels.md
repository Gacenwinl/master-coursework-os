# 投入档位（低 / 中 / 高）

与 **Mode A/B/C**（AI 强度）正交：先选 Mode，再选档位。

| 档位 | 总时长 | 适用 | 裁剪 |
| --- | ---: | --- | --- |
| **Low** | 8–12h | 15% 小 essay；复用熟悉理论 | 单轮 critique；简 literature；无 TikZ |
| **Medium** | 18–25h | 标准 coursework | Galison 实际接近此档 |
| **High** | 35–50h | dissertation 章；双交付；复杂格式 | 多轮 critique；完整 log |

## Medium 默认分配（%）

| Phase | % |
| --- | ---: |
| Intake + Module0 + rubric | 12% |
| Literature + notes | 22% |
| Argument + outline | 18% |
| Draft | 28% |
| Critique + revise | 12% |
| Format + final | 8% |

Low：Draft↑、Literature↓。High：Literature↑、多一轮 critique。
